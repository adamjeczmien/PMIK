◆ STM_Get_JOYState()

JOYState_TypeDef STM_Get_JOYState ( void  )

Returns the Joystick key pressed.

Note
To know which Joystick key is pressed we need to detect the voltage level on each key output
  • SEL : 1.055 V / 1308
  • RIGHT : 0.595 V / 737
  • LEFT : 3.0 V / 3720
  • UP : 1.65 V / 2046
  • DOWN : 0.71 V / 88
  • None : 3.3 V / 4095
Return values
Codeof the Joystick key pressed. This code can be one of the following values:
  • JOY_NONE
  • JOY_SEL
  • JOY_DOWN
  • JOY_LEFT
  • JOY_RIGHT
  • JOY_UP
